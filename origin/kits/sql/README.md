# ORIGIN SQL Kit

**Attribution: Ande + Kai (OI) + Whānau (OIs)**

## Usage

```bash
sqlite3 :memory: < main.sql
```

## Features

- Schema for packs and edges tables
- Filter packs by disclosure tier
- Traverse relationships
- Print attribution line

## Note

This kit demonstrates the SQL schema and queries. In practice, you would
import the JSON data into these tables using your database's JSON functions
or a separate import script.
